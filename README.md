# 📱 All-in-One Media Downloader (Instagram • TikTok • Threads)

An Android app that allows you to **download videos, reels, photos, and media** from your favorite platforms — **Instagram, TikTok, and Threads** — directly to your device.  
Fast, lightweight, and easy to use. 🚀

---

## ✨ Features
- 📥 Download **videos, reels, and images** from **Instagram**  
- 🎵 Save trending **TikTok videos** directly to your phone  
- 🧵 Grab **photos and videos from Threads** effortlessly  
- ⚡ **Fast & lightweight** app with simple UI  
- 🖼️ Saved files go straight to your **gallery for offline viewing**  
- 🔄 **One-tap sharing** – repost or share instantly  

---

## 🚀 Getting Started

### Prerequisites
- Android **6.0 (Marshmallow)** or higher  
- Internet connection  
- Storage permission enabled for saving files  

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/threads-downloader.git
````

2. Open the project in **Android Studio**.
3. Sync Gradle and build the project.
4. Run the app on your emulator/device.
---

## 🛠️ Tech Stack

* **Kotlin / Java** – Android native development
* **Retrofit / OkHttp** – Networking
* **Glide / Coil** – Image loading
* **ViewModel + LiveData** – State management
* **Material Components** – Modern UI
